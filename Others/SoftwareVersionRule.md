### Semantic Versioning(SemVer)이란?
* 소스 버전 Number 체계 에 대한 관리 방법
* Nodejs,npm 모두 Semantic Versioning을 따르고 있음.

### 버전의 형태
```java
MAJOR.MINOR.PATCH
```
* MAJOR: API 호환성이 깨질만한 변경사항
* MINOR: 하위 호환성 지키면서 API 기능이 추가된 것
* PATCH: 하위 호환성 지키는 범위 내에서 버그가 수정된 것
